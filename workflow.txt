Merchant / Staff |	Private Mini (merchant dashboard) |	Authorized store operators open a private Snap Mini to see and update orders in real time.

TL;DR – Your Flask Module Stack for Dual-Snap Setup


LAYER	                  MODULE

Web server	         Flask
Database	         Flask-SQLAlchemy (PostgreSQL)
Auth	             Flask-JWT-Extended + Snap Login Kit
Realtime	         Flask-SocketIO (with Redis backend)
Validation	         Marshmallow
Cache / Queue	     Redis
Frontend hosting	 Flask static + Snap Mini web bundle
Optional async	     Celery (for background tasks like notifications)
